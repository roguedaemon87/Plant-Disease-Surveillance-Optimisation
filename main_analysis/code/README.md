# Main Analysis – Code

Code used for the main manuscript analysis.

---

## Subfolders

- **simulations/**  
  Run epidemic spread simulations.

- **optimisation/**  
  Optimise surveillance site locations.

- **figure_generation/**  
  Generate manuscript figures and summary results.

- **nigeria_comparison/**  
  Nigeria comparison analysis.  
  Contains simulation and optimisation workflows used to compare
  Nigerian road networks with those of the DRC.

---

## Inputs

Inputs for the main analysis scripts are located in:

- `main_analysis/host_distributions/`
- `main_analysis/road_patterns/`

Inputs for the Nigeria comparison workflow are located within:

- `main_analysis/code/nigeria_comparison/`

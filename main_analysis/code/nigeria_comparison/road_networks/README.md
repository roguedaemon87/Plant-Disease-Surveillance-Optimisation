Nigeria Road Networks
=====================

This folder contains road network shapefiles for Nigerian states used in the
comparison analysis.

Example filenames:
- roads_Anambra.shp
- roads_Kebbi.shp
- roads_Nasarawa.shp
- roads_Ogun.shp
- roads_Plateau.shp

These shapefiles are used to determine which host grid cells are accessible
for surveillance under road-constrained optimisation.
